# Unity 2D Shooting Systems

A collection of shooting mechanics for 2D games developed in Unity.

## Features

- Single Shot
- Auto Fire
- Burst Fire
- Spread Shot
- Homing Missile
- Laser Beam
- Projectile-based Shooting

## Technologies

- Unity
- C#
- Rigidbody2D
- Prefabs

## Purpose

This project demonstrates different shooting systems commonly used in 2D shooter games and can be used as a learning resource or starting point for your own projects.
